Про цей шаблон

Цей шаблон створено для тих, хто хоче швидко запустити красиву сторінку без складної верстки. Він добре підходить для особистого сайту, сторінки курсу, портфоліо, наукового проєкту або невеликого продуктового лендингу.

Усе оформлення тримається на одному файлі style.css, який можна покласти в корінь репозиторію разом із цим файлом index.html.

Як можна використовувати сторінку

Тут можна публікувати короткі новини, оголошення, анонси занять, опис проєктів, посилання на матеріали або результати досліджень.

Добре оформлена сторінка викликає довіру ще до того, як читач відкриє перше посилання.

Завдяки картковому стилю навіть простий текст виглядає впорядковано і сучасно.

Швидкий старт

Покладіть у корінь репозиторію два файли:

  • index.html
  • style.css

Після цього увімкніть GitHub Pages у налаштуваннях репозиторію.
